# 代理脑实验进展汇报

## 1. 实验目标与总体设计

当前实验围绕层级化代理脑模型（Hierarchical Agentic Brain, HAB）展开，目标是评估“神经信号编码、认知单元抽象、认知世界模型、不确定性估计与风险感知决策”这一完整建模框架在脑机接口任务中的可行性与有效性。

实验设计不仅关注 EEG 分类准确率，也同时关注模型校准、风险决策、世界模型预测能力和认知图结构指标。整体上希望回答以下问题：

| 实验问题 | 评价重点 |
| --- | --- |
| HAB 是否提升脑信号解码性能 | Accuracy、Macro-F1、Kappa |
| 认知单元抽象是否有效 | 与去除认知单元或直接分类模型比较 |
| 世界模型是否有效 | 下一状态预测误差、w/o World Model 消融 |
| 认知图结构是否合理 | 图稀疏性、edge entropy、结构稳定性 |
| 不确定性感知决策是否可靠 | NLL、ECE、Brier、Selective Risk、Expected Cost |

为保证实验结果真实可靠、科学可用和可复现，当前实验遵循以下原则：

1. 数据划分严格避免测试集泄漏，模型选择、阈值选择和校准参数只在训练集或验证集上完成。
2. 所有方法使用一致的数据预处理、任务定义和评价协议，保证对比公平。
3. 除分类指标外，同时报告校准、风险决策、世界模型和图结构指标。
4. 消融实验保持除被删除模块外的其余设置一致，用于判断模块贡献。
5. 保留配置文件、随机种子、日志、每折结果和聚合结果，便于后续复核。
6. 论文结论与真实实验结果保持一致，不根据预期结果反向调整参数或选择性报告。

## 2. 当前已完成工作

### 2.1 小规模验证实验

已完成 PhysioNet EEGBCI 小规模验证实验，主要用于检查数据加载、划分策略、模型训练、指标输出和消融流程是否正常。

该阶段验证了完整训练与评估流程可以正常运行，也暴露出 HAB-Full 在小样本下存在不稳定现象。因此，在后续正式实验中保留了 `HAB-NoWorldModel` 和 `HAB-DenseGraph` 两个关键对照，并对 world-model loss、认知图稀疏结构和辅助损失 warm-up 进行了修正。

### 2.2 PhysioNet EEGMMIDB 全量外部验证

## 3. 当前主要实验结果

PhysioNet 全量实验的主分类结果如下：

| 方法 | Acc | Macro-F1 | Kappa | ECE |
| --- | ---: | ---: | ---: | ---: |
| EEGNet-Lite | 0.806 | 0.798 | 0.613 | 0.122 |
| CSP-LDA | 0.608 | 0.553 | 0.215 | 0.140 |
| Bandpower-LDA | 0.570 | 0.519 | 0.140 | 0.104 |
| HAB-DenseGraph | 0.502 | 0.486 | 0.002 | 0.055 |
| HAB-NoWorldModel | 0.498 | 0.485 | -0.004 | 0.059 |
| HAB-Full | 0.494 | 0.484 | -0.011 | 0.060 |

从当前结果看，EEGNet-Lite 在分类性能上最强，CSP-LDA 和 Bandpower-LDA 也高于当前 HAB 版本。HAB-Full 在 PhysioNet 二分类设置下暂未表现出分类性能优势。

辅助指标方面，HAB 的结构模块仍提供了一定可分析信息：

| 指标 | 观察结果 |
| --- | --- |
| 世界模型 proxy | HAB-Full 的 next-state MSE 优于 copy baseline，说明动态预测任务具有一定有效性 |
| 图结构指标 | HAB-Full 与 DenseGraph 在 edge entropy 和 mean max edge 上存在明显差异，说明稀疏认知图结构被实现并可度量 |
| 校准指标 | HAB 系列 ECE 较低，但需要结合分类性能一起解释 |
| 风险决策 | 当前 HAB-Full 的 selective risk 和 expected cost 尚未优于 EEGNet-Lite 或 CSP-LDA |

## 4. 当前阶段性判断

当前实验可以支持以下阶段性结论：

1. 已建立一套可复现的 HAB 实验流程，覆盖分类、校准、世界模型、认知图和风险决策指标。
2. PhysioNet EEGMMIDB 全量外部验证已完成，实验规模较充分，运行结果完整且无失败行。
3. HAB 的结构性模块可以被实现和量化，尤其是稀疏认知图与稠密图对照之间存在可区分的结构指标。
4. 世界模型辅助任务具有一定动态预测信号，但目前尚未转化为更好的分类性能。

当前结果更适合作为“框架可实现、模块可度量、现有原型仍需改进”的阶段性证据。后续仍需要通过主数据集实验和更完整的消融分析进一步判断 HAB 框架的有效性。

## 5. 当前发现的问题与希望讨论的方向

结合目前 full 实验结果，后续希望重点讨论以下问题：

1. 当前 HAB-Full 在 PhysioNet 二分类设置下分类性能偏弱，主要原因更可能来自模型结构、训练目标、数据任务设置，还是当前 PhysioNet 协议本身不适合作为主要验证场景？
2. HAB-Full 同时优化分类、世界模型、图结构、神经-认知对齐和校准/风险相关目标，多目标联合训练是否可能影响判别表征学习？
3. world-model proxy 已经表现出一定动态预测信号，但尚未带来分类收益。后续是否应将 world-model 目标改为预训练、辅助评估指标，或设计更明确的下一窗口/下一状态预测任务？
4. 当前认知图结构可以被度量，但解释性仍不充分。是否需要引入更明确的生理区域、频带、任务阶段约束，或调整认知单元数量与连接方式？
5. 小世界结构约束是否应继续保留？如果保留，后续应采用哪些更严格的 small-worldness 指标和随机图对照？
6. 在后续 BNCI2014_001 主实验中，是否应先简化 HAB 架构，优先保证分类性能，再逐步加入世界模型、认知图结构约束和风险决策模块？
7. 当前结果是否更适合将论文定位为“代理脑建模框架与诊断性实验”，而不是直接强调“性能显著优越”的模型？

## 6. 后续计划

下一步建议在上述问题明确后，优先补充主数据集实验和更完整的消融分析：

| 优先级 | 后续实验 | 目的 |
| --- | --- | --- |
| 高 | BNCI2014_001 被试内跨会话实验 | 验证模型对跨会话信号漂移的适应能力 |
| 高 | BNCI2014_001 跨被试 LOSO 实验 | 验证新被试泛化能力 |
| 高 | 增加 FBCSP-LDA、ShallowConvNet、DeepConvNet | 补充更强 baseline，保证比较充分 |
| 高 | 增加 Dynamic Neural Graph Only / w/o Cognitive Units | 判断认知单元抽象是否真正有效 |
| 中 | 增加 GRU Predictor 世界模型 baseline | 判断 HAB world model 是否优于普通序列预测模型 |
| 中 | 完善小世界结构指标与随机图参照 | 更严格支撑图结构分析 |
| 中 | 绘制 per-subject、coverage-risk、reliability 和认知图可视化 | 支撑结果解释和论文展示 |

总体上，当前实验已经完成外部鲁棒性验证的主要闭环，但主实验仍需围绕 BNCI2014_001 继续补充。后续具体路线建议结合上述问题的讨论结果进一步确定。
