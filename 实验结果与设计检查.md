## 2. 验证中发现并已修复的问题

### 2.1 World-model loss 的时序定义

问题：早期实现中，world-model loss 可能使用打乱 batch 中相邻样本作为“下一状态”，这不符合真实时序含义。

修复：现在只允许同一被试且原始索引连续的样本对参与 world-model loss。这样世界模型目标才对应连续脑状态预测，而不是 batch 内随机邻居。

影响：该修复提高了实验严谨性，不改变测试集，也不是为了提升某个结果。

### 2.2 认知图结构退化

问题：早期 cognitive adjacency 接近均匀全连接，无法支撑“小世界认知图”或结构解释。

修复：加入 `cognitive_top_k=3`，每个认知单元只保留最相关的 3 条连接。`HAB-DenseGraph` 保留为无稀疏化对照。

验证：修复后，Full HAB 的 `edge_entropy` 从接近 `ln(8)=2.079` 降到约 `1.099`，`mean_max_edge` 从约 `0.125` 上升到约 `0.334`，说明认知图不再是均匀全连接。

### 2.3 HAB-Full 多目标训练不稳定

问题：Full HAB 同时优化分类、world-model、神经-认知对齐和小世界结构，早期训练阶段不同目标可能互相拉扯。

修复：加入辅助损失 warm-up：

| 参数 | 设置 | 含义 |
| --- | --- | --- |
| `aux_warmup_epochs` | 5 | 前 5 个 epoch 只训练主分类目标 |
| `aux_ramp_epochs` | 10 | 接下来 10 个 epoch 逐渐加入辅助损失 |

该策略是多目标训练的稳定化处理。它不改变数据、不碰测试集、不针对某个测试结果调参。

### 2.4 不确定性校准

问题：论文包含不确定性感知和风险决策模块，如果直接使用未校准概率，ECE、Brier、selective risk 和 expected cost 可能不稳定。

修复：加入 validation-only temperature scaling。每个 method/fold/seed 只在 validation split 上拟合一个 temperature，然后用于 test probability calibration。分类 Acc/Macro-F1/Kappa 不因 temperature scaling 改变；NLL/ECE/Brier 和风险决策使用 calibrated probability，同时保留 `raw_nll`、`raw_ece`、`raw_brier` 作为对照。

原则：temperature scaling 不使用 test label，不为了追求测试结果调参，是标准的后验校准步骤，符合可信决策实验设计。

## 5. 后续结果判读原则

正式结果出来后，应按以下顺序判断，而不是只看 HAB-Full accuracy：

1. 主分类性能：比较 Acc、Macro-F1、Kappa。
2. 稳定性：看不同被试和不同 seed 的标准差。
3. 校准性能：比较 NLL、ECE、Brier。
4. 校准前后差异：比较 `raw_nll/raw_ece/raw_brier` 与 `nll/ece/brier`。
5. 风险决策：看相同 coverage 下 selective risk 和 expected cost。
6. 世界模型：看 HAB-Full 是否比 HAB-NoWorldModel 有更好的 next-state proxy，且是否不显著牺牲分类。
7. 图结构：看 HAB-Full 与 HAB-DenseGraph 的 graph metrics 是否体现稀疏结构优势。
8. 统计显著性：全量后再做 paired test，避免小样本验证阶段下结论。

论文中可以支撑的结论必须和结果对应：

| 可能结论 | 所需证据 |
| --- | --- |
| HAB 改善分类 | HAB-Full 或合理 HAB 变体在全量 Acc/Macro-F1/Kappa 上优于强 baseline |
| 世界模型有用 | HAB-Full 相比 HAB-NoWorldModel 在 next-state proxy 或风险/泛化上有优势 |
| 稀疏认知图有用 | HAB-Full 相比 HAB-DenseGraph 有更好结构指标或更稳泛化 |
| 风险策略有效 | selective risk / expected cost 在相同 coverage 下更低 |
| 结构可解释 | cognitive graph 不退化为均匀全连接，并有稳定图指标 |

可以支撑：

1. 实验流程真实可靠：109 被试、3 seeds、1962 行主结果全部成功，无 failed 行。
2. 代理脑框架可以被实现为包含分类、世界模型、认知图和风险决策的统一实验系统。
3. 稀疏认知图和 dense graph 对照具有可区分的结构指标，说明图结构分析不是空的。
4. 世界模型 proxy 在 HAB-Full 中相对 copy baseline 有一定预测收益。
5. 校准与风险指标可以作为 BCI 代理决策系统的补充评价维度。

不能支撑：

1. 不能声称 HAB-Full 分类性能优于 EEGNet-Lite、CSP-LDA 或 Bandpower-LDA。
2. 不能声称世界模型显著提升分类准确率。
3. 不能声称当前小世界/认知图设计已经带来更强泛化。
4. 不能把低 ECE 单独解释为模型更好，因为 HAB 系列接近随机分类，概率校准好不等于判别能力强。

建议论文写法从“提出一个性能优越的代理脑模型”调整为“提出一个可复现实验框架，并发现当前代理脑结构在 PhysioNet MI 任务上尚未超过成熟 EEGNet/CSP baseline；进一步分析显示结构模块可被度量，但其判别能力仍需改进”。这样结果是真实、科学、可用的，也更容易经得起审稿质疑。
